# Frontend Wizards — Brand Voice

## Two core theses (never contradict these)

1. **IDENTITY:** "Every front-end developer is a wizard."
2. **POWER:** "Agentic coding is the source of that magic — the power to conjure whole interfaces from plain words."

## Framing device
The page is the initiation scroll of a secret order, the Frontend Wizards. The visitor arrives an ordinary developer and is shown, section by section, that they are already "one of us."

## Tone principles
- Confident, never arrogant — state the power as fact, let demos carry the weight
- Playful-but-polished — wizardry metaphors map precisely to real things
- Economical — one strong sentence over two average ones
- Conspiratorial and inviting — address the reader as a peer nearing a threshold
- Grounded in demonstration — every claim ties to something visible on the page

## On-brand lexicon
conjure, incantation, summon, ward, the order, one of us, apprentice→wizard, agents, prompts, grimoire, spell, cast, reveal, true form

## Words to avoid
leverage, solutions, platform, synergy, AI-powered (cliché), unlock, game-changer, revolutionise, seamless, robust, scalable (in marketing), innovative, cutting-edge

## Section narrative arc
HERO (summons) → TRUE FORM (identity revealed) → CONJURE (power demonstrated) → ICE WALL (mastery — warding chaos) → ARCADE (play/proof) → REVEAL (both theses land; welcome to the order)

**Why:** Established during full copy pass on 2026-06-15.
**How to apply:** All new copy for this project must advance these theses and use this lexicon. Check the arc if writing a new section.
